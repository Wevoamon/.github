# WN Programming Language

WN is a modern systems programming language developed by the Wevoamon project.

Our goal is to build powerful native software across multiple platforms.

## Core Projects

- WN Compiler
- WN Runtime
- WN Standard Library
- WN Developer Tools

## Philosophy

• Simplicity  
• Performance  
• Native development
